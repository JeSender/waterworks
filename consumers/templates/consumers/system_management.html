What do I do with these files?

This will download a ZIP archive containing .json lists (Django "Fixtures"). These text files contain highly precise, structured backups of every record. If the system ever crashes or data gets deleted, your developer can easily run a command (python manage.py loaddata) to restore exactly what is in this zip file!

You can download records from the same month multiple times. The system generates a fresh file with the exact current timestamp.
